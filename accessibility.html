אודות הנגישות באתר

לאה בודיק – אדריכלות ועיצוב פנים מחויבת להנגשת האתר לאנשים עם מוגבלויות, בהתאם לחוק שוויון זכויות לאנשים עם מוגבלות, התשנ"ח-1998, ולתקנות שוויון זכויות לאנשים עם מוגבלות (התאמות נגישות לשירות), התשע"ג-2013.

תקן ישראלי 5568
WCAG 2.0 AA

האתר תואם את דרישות התקן הישראלי (ת"י 5568) המבוסס על הנחיות הנגישות לתכני אינטרנט WCAG 2.0 ברמת AA.

התאמות הנגישות שבוצעו באתר

מבנה ונווּט

  • האתר בנוי במבנה סמנטי תקני (HTML5) עם שימוש נכון בתגיות header, nav, main, section, footer
  • כל דף מכיל כותרת ראשית (h1) אחת בלבד, והיררכיית כותרות (h2, h3) לוגית
  • קיים קישור "דלג לתוכן המרכזי" בראש כל דף
  • תפריט ניווט נגיש עם תוויות ARIA מתאימות
  • כל הקישורים מנוסחים באופן ברור ומתאר את יעדם

מקלדת ופוקוס

  • ניתן לנווט בכל האתר באמצעות מקלדת בלבד (Tab, Shift+Tab, Enter, Space, Escape)
  • סדר הפוקוס תואם את הסדר הוויזואלי
  • כל האלמנטים האינטראקטיביים מקבלים אינדיקציית פוקוס ברורה
  • תפריט הנייד ניתן לפתיחה וסגירה במקלדת, כולל מקש Escape
  • תיבת התמונות המוגדלות (Lightbox) נגישה במלואה למקלדת

תצוגה וקריאות

  • יחס ניגודיות צבעים עומד בדרישות WCAG AA (לפחות 4.5:1 לטקסט רגיל)
  • האתר מותאם להגדלה של עד 200% ללא אובדן תוכן או פונקציונליות
  • גופנים ברורים וקריאים בגדלים מותאמים
  • האתר תומך בהעדפות מערכת לצמצום תנועה (prefers-reduced-motion)
  • האתר מותאם לכיוון קריאה מימין לשמאל (RTL) בעברית

תמונות ומדיה

  • כל התמונות כוללות תיאור חלופי (alt) משמעותי בעברית
  • תמונות דקורטיביות מסומנות כך שקוראי מסך יתעלמו מהן
  • גלריות התמונות נגישות למקלדת ולקוראי מסך

טפסים

  • כל שדות הקלט מקושרים לתוויות (labels) מזהות
  • הודעות שגיאה מוצגות בטקסט ברור ומקושרות לשדה הרלוונטי
  • שדות חובה מסומנים בבירור

תכנים מיוחדים

  • קרוסלת ההמלצות כוללת כפתורי ניווט נגישים
  • אזורים דינמיים מסומנים עם ARIA live regions
  • אייקונים מסומנים ב-aria-hidden כאשר יש להם טקסט מלווה

טכנולוגיות נגישות

האתר נבנה ונבדק באמצעות:

  • HTML5 סמנטי
  • CSS3 עם משתני עיצוב (CSS Custom Properties)
  • תכונות ARIA (Accessible Rich Internet Applications)
  • JavaScript ללא תלות בספריות חיצוניות

דפדפנים נתמכים

האתר נבדק ותואם לדפדפנים:

  • Google Chrome (גרסה עדכנית)
  • Mozilla Firefox (גרסה עדכנית)
  • Microsoft Edge (גרסה עדכנית)
  • Apple Safari (גרסה עדכנית)
  • דפדפנים ניידים: Chrome Mobile, Safari Mobile

מסמכים

האתר אינו מכיל קבצי PDF או מסמכים להורדה. כל התוכן מוצג בפורמט HTML נגיש.
